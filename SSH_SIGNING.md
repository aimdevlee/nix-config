# SSH Commit Signing Setup

GitHub and GitLab now support SSH keys for commit signing, which is simpler than GPG.

## 1. Check Your SSH Key

```bash
# Check if you have an SSH key
ls -la ~/.ssh/id_ed25519.pub

# If not, generate one
ssh-keygen -t ed25519 -C "aimdevlee@gmail.com"
```

## 2. Add SSH Key to GitHub

### For Authentication (if not already done)
```bash
# Copy your SSH public key
cat ~/.ssh/id_ed25519.pub | pbcopy

# Add to GitHub: Settings → SSH and GPG keys → New SSH key
# Key type: Authentication key
```

### For Signing
```bash
# Copy your SSH public key (same key)
cat ~/.ssh/id_ed25519.pub | pbcopy

# Add to GitHub: Settings → SSH and GPG keys → New SSH key
# Key type: Signing key
```

## 3. Enable SSH Signing in Nix Configuration

Already configured in `home.nix`:
```nix
modules = {
  user = {
    enable = true;
    enableGitSigning = true;  # This enables SSH signing
    # sshSigningKey = "~/.ssh/id_ed25519.pub";  # Default path
  };
  # ... rest of modules
};
```

## 4. Apply Configuration

```bash
sudo darwin-rebuild switch
```

## 5. Verify Configuration

```bash
# Check Git config
git config --list | grep -E "gpg|sign"

# Should show:
# commit.gpgsign=true
# tag.gpgsign=true
# gpg.format=ssh
# user.signingkey=~/.ssh/id_ed25519.pub
# gpg.ssh.allowedSignersFile=~/.ssh/allowed_signers
```

## 6. Test Signed Commit

```bash
# Make a test commit
git commit --allow-empty -m "test: SSH signed commit"

# Verify the signature
git log --show-signature -1

# Push to GitHub
git push

# Check on GitHub - commit should show "Verified" badge
```

## Troubleshooting

### If commits aren't being signed:
```bash
# Manually sign a commit
git commit -S -m "message"

# Check SSH agent
ssh-add -l

# Add key to agent if needed
ssh-add ~/.ssh/id_ed25519
```

### To verify signatures locally:
```bash
# The allowed_signers file is auto-generated by Nix
cat ~/.ssh/allowed_signers
```

## Benefits over GPG
- No passphrase prompts (uses SSH agent)
- Simpler setup
- Uses existing SSH infrastructure
- Works seamlessly with GitHub/GitLab
